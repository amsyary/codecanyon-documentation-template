Installation Process

  1. Upload the VenManager-V1.0.1.zip file to the hosting server.
  2. Once uploaded, extract it to your desired location.
  3. Create a database and import public/db/database.sql.
  4. Open the extracted directory and locate the .env file in the root.
Can't find the .env file?
  1. Open your File Manager settings.
  2. Tick Show Hidden Files (dotfiles).
  3. The .env file will now be visible.
File Manager settings
Tick Show Hidden Files
.env file visible
  1. Open the .env file and update DB_DATABASE, DB_USERNAME, and DB_PASSWORD.
  2. That's it!
